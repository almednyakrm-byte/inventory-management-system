# نظام إدارة مخزونات ومخازن (تسجيل، إدارة، استعراض)
==============================================

## Overview & Project Purpose
---------------

نظام إدارة مخزونات ومخازن هو مشروع يهدف إلى توفير حل شامل لإدارة المخزونات والمخازن. يسمح النظام للمستخدمين بتسجيل وإدارة واستعراض المخزونات بسهولة ويسر. يهدف المشروع إلى توفير أداة فعالة وسهلة الاستخدام للمستخدمين لتحسين إدارة المخزونات والمخازن.

## Project Structure Mapping
-------------------------

* `app`: تطبيق النظام
	+ `models`: نماذج البيانات
	+ `views`: مناظير التطبيق
	+ `controllers`: مراقبة التطبيق
	+ `templates`: قوالب التطبيق
* `config`: إعدادات النظام
* `docker`: ملفات الدوكر
* `requirements`: متطلبات النظام

## Running the Environment
-------------------------

لتشغيل النظام، اتبع الخطوات التالية:

1. تأكد من تثبيت Docker و Docker Compose على جهازك.
2. افتح Terminal وانتقل إلى دليل المشروع.
3. قم بتشغيل الأمر التالي: `docker-compose up -d`
4. انتظر حتى يتم تشغيل جميع الخدمات.
5. افتح متصفحك وانتقل إلى الرابط: `http://localhost:8000`

## Modules, Tables, and Roles
---------------------------

* **Modules**:
	+ تسجيل المخزونات
	+ إدارة المخزونات
	+ استعراض المخزونات
* **Tables**:
	+ مخزونات
	+ فئات المخزونات
	+ مواصفات المخزونات
* **Roles**:
	+ مدير النظام
	+ مشرف المخزونات
	+ مستخدم المخزونات
